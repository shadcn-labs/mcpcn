## Usage

```tsx
import {
  YouTubePost,
  YouTubePostAvatar,
  YouTubePostDetails,
  YouTubePostInfo,
  YouTubePostMenu,
  YouTubePostPlayer,
} from "@/components/ui/youtube-post";
```

```tsx
<YouTubePost>
  <YouTubePostPlayer />
  <YouTubePostInfo>
    <div className="flex gap-3">
      <YouTubePostAvatar />
      <YouTubePostDetails />
      <YouTubePostMenu />
    </div>
  </YouTubePostInfo>
</YouTubePost>
```

## Composition

Use the following composition to build an `YouTubePost`:

```text
YouTubePost
├── YouTubePostPlayer
└── YouTubePostInfo
    ├── YouTubePostAvatar
    ├── YouTubePostDetails
    └── YouTubePostMenu
```

## OpenUI Integration

Register `YouTubePost` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import {
  YouTubePost,
  YouTubePostAvatar,
  YouTubePostDetails,
  YouTubePostInfo,
  YouTubePostMenu,
  YouTubePostPlayer,
} from "@/components/ui/youtube-post";

const OpenUIYouTubePost = defineComponent({
  name: "YouTubePost",
  description: "YouTube video card with playable embed.",
  props: z.object({}),
  component: () => (
    <YouTubePost>
      <YouTubePostPlayer />
      <YouTubePostInfo>
        <div className="flex gap-3">
          <YouTubePostAvatar />
          <YouTubePostDetails />
          <YouTubePostMenu />
        </div>
      </YouTubePostInfo>
    </YouTubePost>
  ),
});

export const youtube_postLibrary = createLibrary({
  root: "YouTubePost",
  components: [OpenUIYouTubePost],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
