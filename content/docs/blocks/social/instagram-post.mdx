## Integrate with OpenUI

Register `InstagramPost` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { InstagramPost } from "@/components/ui/instagram-post";

const OpenUIInstagramPost = defineComponent({
  name: "InstagramPost",
  description: "Instagram Post for an MCP-powered interface.",
  props: z.object({
    // Add flat, serializable props that the model should generate.
  }),
  component: ({ props }) => <InstagramPost />,
});

export const instagrampostLibrary = createLibrary({
  root: "InstagramPost",
  components: [OpenUIInstagramPost],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
