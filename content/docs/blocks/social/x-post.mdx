## Integrate with OpenUI

Register `XPost` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { XPost } from "@/components/ui/x-post";

const OpenUIXPost = defineComponent({
  name: "XPost",
  description: "X (Twitter) post card with engagement metrics.",
  props: z.object({}),
  component: () => <XPost />,
});

export const x_postLibrary = createLibrary({
  root: "XPost",
  components: [OpenUIXPost],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
