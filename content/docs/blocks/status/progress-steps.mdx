## Integrate with OpenUI

Register `ProgressSteps` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { ProgressSteps } from "@/components/ui/progress-steps";

const OpenUIProgressSteps = defineComponent({
  name: "ProgressSteps",
  description:
    "Progress indicator with horizontal or vertical layout and step statuses.",
  props: z.object({}),
  component: () => <ProgressSteps />,
});

export const progress_stepsLibrary = createLibrary({
  root: "ProgressSteps",
  components: [OpenUIProgressSteps],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
