## Integrate with OpenUI

Register `MapCarousel` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { MapCarousel } from "@/components/ui/map-carousel";

const OpenUIMapCarousel = defineComponent({
  name: "MapCarousel",
  description:
    "Interactive map with location markers and a draggable carousel of cards.",
  props: z.object({}),
  component: () => <MapCarousel />,
});

export const map_carouselLibrary = createLibrary({
  root: "MapCarousel",
  components: [OpenUIMapCarousel],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
