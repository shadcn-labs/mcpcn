## Integrate with OpenUI

Register `TicketTierSelect` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { TicketTierSelect } from "@/components/ui/ticket-tier-select";

const OpenUITicketTierSelect = defineComponent({
  name: "TicketTierSelect",
  description: "Ticket Tier Select for an MCP-powered interface.",
  props: z.object({
    currency: z.string().default("USD"),
    selectedTier: z.string().optional(),
  }),
  component: ({ props }) => <TicketTierSelect {...props} />,
});

export const tickettierselectLibrary = createLibrary({
  root: "TicketTierSelect",
  components: [OpenUITicketTierSelect],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
