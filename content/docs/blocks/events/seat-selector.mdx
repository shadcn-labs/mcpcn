## Integrate with OpenUI

Register `SeatSelector` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { SeatSelector } from "@/components/ui/seat-selector";

const OpenUISeatSelector = defineComponent({
  name: "SeatSelector",
  description: "Seat Selector for an MCP-powered interface.",
  props: z.object({
    selectedSeats: z.array(z.string()).default([]),
  }),
  component: ({ props }) => <SeatSelector {...props} />,
});

export const seatselectorLibrary = createLibrary({
  root: "SeatSelector",
  components: [OpenUISeatSelector],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
