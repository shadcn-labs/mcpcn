## OpenUI Integration

Register `OptionList` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { OptionList } from "@/components/ui/option-list";

const OpenUIOptionList = defineComponent({
  name: "OptionList",
  description:
    "Tag-style option selector with single or multiple selection modes.",
  props: z.object({}),
  component: () => <OptionList />,
});

export const option_listLibrary = createLibrary({
  root: "OptionList",
  components: [OpenUIOptionList],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
