## OpenUI Integration

Register `QuickReply` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { QuickReply, QuickReplyList } from "@/components/ui/quick-reply";

const OpenUIQuickReply = defineComponent({
  name: "QuickReply",
  description: "Quick reply buttons for common chat responses.",
  props: z.object({}),
  component: () => (
    <QuickReply>
      <QuickReplyList />
    </QuickReply>
  ),
});

export const quick_replyLibrary = createLibrary({
  root: "QuickReply",
  components: [OpenUIQuickReply],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
