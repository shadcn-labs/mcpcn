## Integrate with OpenUI

Register `OrderStatus` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { OrderStatus } from "@/components/ui/order-status";

const OpenUIOrderStatus = defineComponent({
  name: "OrderStatus",
  description: "Order Status for an MCP-powered interface.",
  props: z.object({
    // Add flat, serializable props that the model should generate.
  }),
  component: ({ props }) => <OrderStatus />,
});

export const orderstatusLibrary = createLibrary({
  root: "OrderStatus",
  components: [OpenUIOrderStatus],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
