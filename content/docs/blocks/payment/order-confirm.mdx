## Integrate with OpenUI

Register `OrderConfirm` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { OrderConfirm } from "@/components/ui/order-confirm";

const OpenUIOrderConfirm = defineComponent({
  name: "OrderConfirm",
  description: "Order Confirm for an MCP-powered interface.",
  props: z.object({
    orderNumber: z.string().optional(),
    currency: z.string().default("USD"),
  }),
  component: ({ props }) => <OrderConfirm {...props} />,
});

export const orderconfirmLibrary = createLibrary({
  root: "OrderConfirm",
  components: [OpenUIOrderConfirm],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
