## Integrate with OpenUI

Register `AmountInput` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { AmountInput } from "@/components/ui/amount-input";

const OpenUIAmountInput = defineComponent({
  name: "AmountInput",
  description:
    "Amount input with increment/decrement buttons and preset values.",
  props: z.object({}),
  component: () => <AmountInput />,
});

export const amount_inputLibrary = createLibrary({
  root: "AmountInput",
  components: [OpenUIAmountInput],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
