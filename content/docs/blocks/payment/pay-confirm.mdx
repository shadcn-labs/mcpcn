## Integrate with OpenUI

Register `PayConfirm` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { PayConfirm } from "@/components/ui/pay-confirm";

const OpenUIPayConfirm = defineComponent({
  name: "PayConfirm",
  description: "Pay Confirm for an MCP-powered interface.",
  props: z.object({
    currency: z.string().default("USD"),
    isLoading: z.boolean().default(false),
  }),
  component: ({ props }) => <PayConfirm {...props} />,
});

export const payconfirmLibrary = createLibrary({
  root: "PayConfirm",
  components: [OpenUIPayConfirm],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
