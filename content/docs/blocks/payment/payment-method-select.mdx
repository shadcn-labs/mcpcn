## Integrate with OpenUI

Register `PaymentMethodSelect` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { PaymentMethodSelect } from "@/components/ui/payment-method-select";

const OpenUIPaymentMethodSelect = defineComponent({
  name: "PaymentMethodSelect",
  description: "Payment Method Select for an MCP-powered interface.",
  props: z.object({
    name: z.string().default("payment-method"),
    selectedMethod: z.string().optional(),
  }),
  component: ({ props }) => <PaymentMethodSelect {...props} />,
});

export const paymentmethodselectLibrary = createLibrary({
  root: "PaymentMethodSelect",
  components: [OpenUIPaymentMethodSelect],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
