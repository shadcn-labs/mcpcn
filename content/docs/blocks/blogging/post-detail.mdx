## Integrate with OpenUI

Register `PostDetail` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { PostDetail } from "@/components/ui/post-detail";

const OpenUIPostDetail = defineComponent({
  name: "PostDetail",
  description:
    "Full post view with cover image, author info, content, tags and related posts section.",
  props: z.object({}),
  component: () => <PostDetail />,
});

export const post_detailLibrary = createLibrary({
  root: "PostDetail",
  components: [OpenUIPostDetail],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
