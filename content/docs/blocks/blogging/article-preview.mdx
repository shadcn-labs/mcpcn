## Integrate with OpenUI

Register `ArticlePreview` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { ArticlePreview } from "@/components/ui/article-preview";

const OpenUIArticlePreview = defineComponent({
  name: "ArticlePreview",
  description: "Article Preview for an MCP-powered interface.",
  props: z.object({
    // Add flat, serializable props that the model should generate.
  }),
  component: ({ props }) => <ArticlePreview />,
});

export const articlepreviewLibrary = createLibrary({
  root: "ArticlePreview",
  components: [OpenUIArticlePreview],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
