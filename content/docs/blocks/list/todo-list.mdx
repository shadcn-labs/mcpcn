## Integrate with OpenUI

Register `TodoList` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { TodoList } from "@/components/ui/todo-list";

const OpenUITodoList = defineComponent({
  name: "TodoList",
  description: "Todo List for an MCP-powered interface.",
  props: z.object({
    // Add flat, serializable props that the model should generate.
  }),
  component: ({ props }) => <TodoList />,
});

export const todolistLibrary = createLibrary({
  root: "TodoList",
  components: [OpenUITodoList],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
