## Integrate with OpenUI

Register `ProductList` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { ProductList } from "@/components/ui/product-list";

const OpenUIProductList = defineComponent({
  name: "ProductList",
  description: "Product List for an MCP-powered interface.",
  props: z.object({
    currency: z.string().default("USD"),
  }),
  component: ({ props }) => <ProductList {...props} />,
});

export const productlistLibrary = createLibrary({
  root: "ProductList",
  components: [OpenUIProductList],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
