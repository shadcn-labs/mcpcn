## Integrate with OpenUI

Register `Table` with `defineComponent`, describe its model-generated props with Zod, and include it in a library.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { Table } from "@/components/ui/table";

const OpenUITable = defineComponent({
  name: "Table",
  description:
    "Data table with optional single or multi-select modes for chat interfaces.",
  props: z.object({}),
  component: () => <Table />,
});

export const tableLibrary = createLibrary({
  root: "Table",
  components: [OpenUITable],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
