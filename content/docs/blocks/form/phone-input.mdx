## Integrate with OpenUI

Register `PhoneInput` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { PhoneInput } from "@/components/ui/phone-input";

const OpenUIPhoneInput = defineComponent({
  name: "PhoneInput",
  description: "Phone Input for an MCP-powered interface.",
  props: z.object({
    defaultCountry: z.string().default("US"),
    defaultValue: z.string().default(""),
  }),
  component: ({ props }) => <PhoneInput {...props} />,
});

export const phoneinputLibrary = createLibrary({
  root: "PhoneInput",
  components: [OpenUIPhoneInput],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
