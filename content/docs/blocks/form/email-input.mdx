## Integrate with OpenUI

Register `EmailInput` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { EmailInput } from "@/components/ui/email-input";

const OpenUIEmailInput = defineComponent({
  name: "EmailInput",
  description: "Email Input for an MCP-powered interface.",
  props: z.object({
    defaultValue: z.string().default(""),
    invalidMessage: z.string().default("Enter a valid email address."),
  }),
  component: ({ props }) => <EmailInput {...props} />,
});

export const emailinputLibrary = createLibrary({
  root: "EmailInput",
  components: [OpenUIEmailInput],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
