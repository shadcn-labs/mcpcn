## Integrate with OpenUI

Register `RatingCard` with `defineComponent`, describe its model-generated props with Zod, and include it in a library. Keep the schema flat so OpenUI Lang can generate and stream it reliably.

```tsx showLineNumbers
import { createLibrary, defineComponent } from "@openuidev/react-lang";
import { z } from "zod/v4";

import { RatingCard } from "@/components/ui/rating-card";

const OpenUIRatingCard = defineComponent({
  name: "RatingCard",
  description: "Rating Card for an MCP-powered interface.",
  props: z.object({
    defaultRating: z.number().min(0).max(5).default(0),
  }),
  component: ({ props }) => <RatingCard {...props} />,
});

export const ratingcardLibrary = createLibrary({
  root: "RatingCard",
  components: [OpenUIRatingCard],
});
```

For richer generated compositions, define the child primitives separately and connect them with `.ref` fields. See [Defining Components](https://www.openui.com/docs/openui-lang/defining-components) in the OpenUI documentation.
